# Random Task Generator — Генератор случайных задач

Графическое приложение для генерации случайных задач с фильтрацией по типу,
сохранением истории и тестами.

**Автор: Рязанцев Егор**

---

## Возможности

- База из 20 предопределённых задач (учёба, спорт, работа)
- Кнопка «Сгенерировать задачу» — выбирает случайную задачу с учётом фильтра
- Фильтрация по типу задачи
- Кнопка «Добавить свою задачу» с валидацией (пустая строка не принимается)
- История сгенерированных задач в виде списка
- Сохранение истории в `history.json` и загрузка при следующем запуске

---

## Структура проекта

```
task-generator/
├── main.py          # GUI-приложение (tkinter)
├── tasks.py         # База предопределённых задач
├── storage.py       # Сохранение / загрузка истории (JSON)
├── tests.py         # Юнит-тесты (unittest)
├── requirements.txt
└── .gitignore
```

---

## Установка и запуск

### Windows

```bash
python main.py
```

### Linux

```bash
sudo apt install python3-tk
python3 main.py
```

---

## Запуск тестов

```bash
python -m unittest tests -v
```

12 тестов покрывают:
- корректность базы задач
- логику фильтрации
- валидацию пустого ввода
- сохранение / загрузку JSON

---

## Примеры использования

1. Запустить `python main.py`
2. Выбрать фильтр (или оставить «Все»)
3. Нажать **Сгенерировать задачу** — задача отобразится в блоке выше
4. История всех генераций сохраняется в списке внизу
5. Добавить свою задачу через **Добавить свою задачу**
